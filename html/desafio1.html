Fatos sobre html

A seguir algumas caracteristicas de html

  • Não é uma linguagem de programação
  • Espaços em texto escrito não necessariamente será visualizado como espaço na pagina
  • Quebras de linhas apenas usando tags especificas, como "p" e "br"
  • O nome da tag "a" vem de "anchor", ou "ãncora"
  • O atributo "target=_blank" da tag "a" abre o link em uma nova janela
